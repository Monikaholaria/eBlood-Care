eBlood Care System

Project Overview

The eBlood Care System is a blood donation and blood request management application developed in C programming language. The system helps connect blood donors and patients by providing blood search and request facilities.

Features

- Donor Registration
- Blood Search
- Blood Request Management
- Display Donor Details
- User-Friendly Menu Interface

Modules

1. Donor Module

Stores donor information such as name, blood group, and contact number.

2. Patient Module

Allows patients to request blood based on their requirements.

3. Blood Search Module

Searches for available donors according to blood group.

4. Blood Request Module

Manages blood requests from patients.

5. Admin Module

Monitors donor records and system activities.

Technologies Used

- Programming Language: C
- Operating System: Windows
- Compiler: GCC / Turbo C / CodeBlocks

System Requirements

Hardware Requirements

- Processor: Intel Core i3 or above
- RAM: 4 GB or above
- Storage: 100 MB free space

Software Requirements

- Windows Operating System
- C Compiler
- CodeBlocks / Dev-C++ / Turbo C

Future Scope

- Online donor database
- Hospital integration
- SMS and email notifications
- Mobile application support

Conclusion

The eBlood Care System provides a simple and efficient way to manage blood donors and blood requests, helping patients find blood quickly during emergencies.
